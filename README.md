# Avelia Luxury | AI-Powered E-Commerce Platform

A high-end, full-stack e-commerce application featuring a minimalist luxury aesthetic and an integrated AI Concierge.

## 🚀 Live Demo
https://avelia-fashion-752376492797.us-central1.run.app

## ✨ Key Features
* **AI Concierge:** A FastAPI-powered chat assistant that provides real-time information about luxury materials and brand inquiries.
* **Dynamic Product Catalog:** Asynchronous product rendering with automated cart management.
* **Cloud-Native Architecture:** Fully containerized with Docker and deployed using a CI/CD pipeline on Google Cloud Run.
* **Luxury UI/UX:** A bespoke frontend built with HTML5, CSS3, and Vanilla JavaScript, optimized for the "Modern Elite."

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Properties), JavaScript (ES6+)
* **Backend:** Python 3.11, FastAPI, Uvicorn
* **DevOps:** Docker, GitHub Actions, Google Cloud Build, Google Cloud Run

## 📂 Project Structure
- `backend/`: FastAPI application logic and API endpoints.
- `frontend/`: UI components, styles, and client-side logic.
- `Dockerfile`: Containerization instructions for cloud deployment.
